# Project Summary

## Urban Air Pollution Analysis & Public Health Risk Assessment

This project investigates seasonal NO₂ pollution dynamics and environmental risk patterns using the UCI Air Quality Dataset.

## Key Highlights

- Analyzed 6,939 hourly air-quality observations
- Identified winter as the season with the highest mean NO₂ concentration
- Estimated probability of extreme NO₂ pollution events
- Applied Wilcoxon rank-sum testing and effect size analysis
- Used Spearman correlation to study atmospheric relationships
- Built a multiple linear regression model explaining approximately 60.9% of NO₂ variability
