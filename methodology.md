# Methodology

This project follows a structured environmental analytics workflow for urban air-quality assessment.

## Analytical Workflow

1. Data import and structure inspection
2. Missing value treatment
3. Date-time parsing and temporal feature engineering
4. Seasonal NO₂ concentration analysis
5. Probability-based high-pollution risk estimation
6. Statistical inference using Wilcoxon rank-sum testing
7. Effect size estimation
8. Spearman correlation analysis
9. Multiple linear regression modeling
10. Output generation and visualization

## Main Target Variable

```text
NO₂ concentration
```

## Extreme Pollution Event Definition

```text
NO₂ > 200
```

## Statistical Methods

- Descriptive statistics
- Probability estimation
- Wilcoxon rank-sum test
- Effect size analysis
- Spearman correlation
- Multiple linear regression
- Regression diagnostics

## Interpretation Focus

The project evaluates seasonal pollution variation, high-risk NO₂ events, atmospheric relationships, and the explanatory power of meteorological and pollutant predictors.
