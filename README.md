# Urban Air Pollution Analysis & Public Health Risk Assessment

Environmental analytics project using the **UCI Air Quality Dataset** to analyze seasonal NO₂ pollution patterns, extreme pollution risk, and atmospheric relationships using R.

## Methods
- Exploratory Data Analysis
- Data Cleaning
- Probability-Based Risk Analysis
- Wilcoxon Rank-Sum Test
- Spearman Correlation
- Multiple Linear Regression

## Key Findings
- Analyzed **6,939 hourly air-quality observations**
- Winter showed the highest mean NO₂ concentration
- Extreme NO₂ pollution risk was highest in winter
- Regression modeling explained about **60.9%** of NO₂ variability
