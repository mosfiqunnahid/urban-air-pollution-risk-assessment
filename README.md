# Data
The project evaluates seasonal pollution variation, high-risk NO₂ events, atmospheric relationships, and the explanatory power of meteorological and pollutant predictor.

Expected filename:

```text
AirQualityUCI.csv
```

The dataset is not included in this repository by default. Download the dataset and place it here before running the analysis script.

Expected format:

- Separator: semicolon (`;`)
- Decimal mark: comma (`,`)
- Missing sensor values may be coded as `-200`
